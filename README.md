# ProtoShell 🐚🐧

ProtoShell is a project for the GNU/Linux 🐧 course we received as trainees to become PROTECO interns.

## Why ProtoShell?
It's an attempt of terminal, so we decided to call it ProtoShell, a subset of an actual terminal.
